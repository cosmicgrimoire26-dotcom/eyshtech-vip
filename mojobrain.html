Setup

How it works

MojoBrain runs on a Mac Mini, Linux box, or virtual machine. Follow the playbook, deploy the agents, and they start working.

1

Deploy the agents

Follow the step-by-step playbook to install each agent. Each one is a Python script with a cron schedule. No Docker, no complex infrastructure. Works on physical hardware or a virtual machine.

2

Configure your Nerve Center

The Nerve Center connects all agents through a shared state file. Define your cross-agent routing rules and which events trigger which responses.

3

Connect Discord

Wire up webhook notifications with channel routing. Critical alerts, routine logs, and agent reports each go where they belong. You see what matters, nothing more.

4

Launch Mission Control

The solarpunk dashboard gives you a real-time view of every agent's status, system health, and alerts. Accessible from any device on your network.

Questions

FAQ

What hardware do I need? +
Any always-on machine: Mac Mini, Linux box, or a virtual machine (cloud VM works too). MojoBrain runs as lightweight Python scripts on cron, so resource needs are minimal. The heavy AI processing happens through your chosen LLM provider's API, not locally.
What are the ongoing costs? +
For MojoBrain: some API keys are usage-based, typically $5-20/mo depending on your volume. We recommend the best LLM provider for your use case during the business evaluation. For CC Jarvis: Claude Max at $100/mo. No per-agent charges either way. The agents run locally as Python scripts that read files, write JSON, and report status.
Is this just an OpenClaw wrapper? +
MojoBrain was built on OpenClaw (open-source) and has evolved into its own architecture with a custom Nerve Center for agent coordination. No gateway to maintain, no framework-specific dependencies. It's agent scripts, cron jobs, JSON state files, and your LLM of choice. Battle-tested in production.
Can I add custom agents? +
Yes. Any Python script that writes a JSON output file can become an agent. The Nerve Center will detect it, track its health, and route its events. The architecture guide (included in Guided Setup and Managed Deploy) walks you through the exact structure.
Do I need coding experience? +
Basic terminal comfort is all you need. The playbook has copy-paste commands for every step. If you can open Terminal and paste a command, you can deploy MojoBrain. For customization, Claude Code helps you modify agents conversationally. No Python expertise required.
How do I get notifications? +
Discord. Each agent reports to its own channel. Critical alerts, routine logs, and agent reports are routed automatically. You see what matters, nothing more. The webhook architecture is simple to set up and the playbook walks you through it.
What industries does this work for? +
Any service business, consultancy, or agency. The Lead Engine and CFO agents are industry-agnostic by design. The CSO and Law agents cover universal security and compliance needs. Guided Setup includes customization to map the Lead Engine to your specific market.
