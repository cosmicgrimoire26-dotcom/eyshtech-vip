FAQ

Common questions.

What hardware do I need? +
Any always-on machine: Mac Mini, Linux box, or a cloud VM. MojoBrain runs as lightweight Python scripts on cron. The heavy AI processing happens through your LLM provider's API, not locally.
What are the ongoing costs? +
For MojoBrain: some API keys are usage-based, typically $5-20/mo depending on volume. For CC Jarvis: Claude Max at $100/mo. No per-agent charges. The agents run locally as Python scripts.
Can I add custom agents? +
Yes. Any Python script that writes a JSON output file can become an agent. The Nerve Center detects it, tracks its health, and routes its events. The architecture guide walks you through the structure.
Do I need coding experience? +
Basic terminal comfort is all you need. The playbook has copy-paste commands for every step. For customization, Claude Code helps you modify agents conversationally. No Python expertise required.
How do I get notifications? +
Discord. Each agent reports to its own channel. Critical alerts, routine logs, and agent reports are routed automatically. You see what matters, nothing more.
What industries does this work for? +
Any service business, consultancy, or agency. The Lead Engine and CFO agents are industry-agnostic. The CSO and Law agents cover universal needs. Guided Setup includes customization to your specific market.
What if I already have tools in place? +
MojoBrain integrates with your existing stack. It connects to Google Workspace, Stripe, Brevo, Discord, and more via API. It doesn't replace your tools — it automates the work between them.
